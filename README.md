# River Water Quality Forecasting System

## Overview

The River Water Quality Forecasting System is a Machine Learning based web application used to analyze and predict river water quality parameters.
This project helps in monitoring water conditions and supports environmental analysis.

## Features

* Water quality prediction
* Data preprocessing and analysis
* Machine Learning model integration
* User-friendly web interface
* Forecasting based on input parameters

## Technologies Used

* Python
* Flask
* Pandas
* NumPy
* Scikit-learn
* HTML
* CSS

## Project Structure

* data/ → Dataset files
* model/ → Trained ML models
* templates/ → HTML pages
* River_Water_Quality_Prediction/ → Main application files

## How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/vanitha00/Water_Quality_Forcasting_System.git
```

2. Install required packages

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open the local server in browser
